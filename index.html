Number Game

How to Play?

  1. Click "Play" to start game.
  2. Enter the number between 1-9.
  3. If the number match with the computer randomly generated number then its draw
  4. If the sum of both the numbers is an even number the “user” (you) are the winner.
  5. If the sum of both numbers is an odd number, then computer is the winner.
  6. Enjoy the game.
